# Volt Consulting Group — voltcg.nl

Marketing site for Volt Consulting Group (VCG).
Vite + React + TypeScript + Tailwind + shadcn/ui, deployed on Vercel.

## Develop
```bash
bun install
bun run dev
```
Dev server: http://localhost:8080

## Build
```bash
bun run build      # production build -> dist/
bun run preview    # preview the build locally
```

## Deploy
Pushes to `main` deploy to production on Vercel. Live domain: https://voltcg.nl

## Brand assets
Favicons, the Open Graph image (`og-image.png`), the web manifest, and the
logo mark (`logo-mark*.svg`) live in `public/`.
Brand colors: red `#FA0A0A` · black `#141414` · white `#FFFFFF`.
